# cursor-setup-project
Setup project using Cursor, Claude Code, and Codex
## Introduction
This project demonstrates my ability to set up development tools and manage a GitHub repository as part of a portfolio assignment.

## Tools Installed
- Cursor IDE
- Claude Code extension
- Codex extension
- GitHub

## Steps Completed
1. Installed Cursor IDE
2. Installed Claude Code extension and logged in
3. Installed Codex extension and logged in
4. Created a public GitHub repository
5. Opened the project in Cursor
6. Edited the README file

## Issues Faced and Solutions
- Issue: Could not find clone repository option in Cursor  
  Solution: Downloaded ZIP from GitHub and opened it manually in Cursor

- Issue: Understanding GitHub connection  
  Solution: Learned how to initialize Git and push code using terminal

## Conclusion
This project helped me understand how to connect development tools with GitHub and manage version control.
